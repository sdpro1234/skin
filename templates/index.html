Instructions

How to Use the Skin Disease Classifier App

  1. Input Data:
    • Text: Type a description of the skin condition (e.g., "Red, itchy rash on the forearm").
    • Image: Upload a clear image of the affected skin area.
  2. Choose a Prediction Mode:
    • Text Prediction: Uses your text input to predict the condition.
    • Image Prediction: Uses your uploaded image.
    • Combined Prediction: Leverages both text and image inputs for a richer prediction.
  3. View Results:
    The app displays the predicted condition along with a confidence chart showing the probability for each possible class. Remedy, medicine, and hospital suggestions for the predicted condition are generated automatically.
  4. Reset:
    Use the Reset Inputs button to clear your inputs and start over.

Note: This tool is meant for informational purposes only and should not be used as a substitute for professional medical advice.

Overview

This web app leverages state-of-the-art deep learning models to perform multimodal analysis of skin conditions.

  • Text Analysis: A BART-based model processes descriptions.
  • Image Analysis: A ResNet18 model (pre-trained on ImageNet) extracts visual features.
  • Combined Analysis: Integrates both text and image modalities for enhanced predictions.
